# Alacritty - Terminal Emulator

> **Status: Legacy** — Replaced by [Ghostty](../ghostty/README.md) as the primary terminal.

Alacritty was the original GPU-accelerated terminal in this setup. The configuration is kept for reference.

## Configuration Summary

| Setting | Value |
|---------|-------|
| Theme | Rose Pine |
| Font | JetBrainsMono Nerd Font, 18pt |
| Opacity | 0.75 |
| Cursor | Block, non-blinking |
| Decorations | Buttonless |
| `option_as_alt` | Both |

## Keybindings

| Key | Action |
|-----|--------|
| `Cmd+Shift+R` | Reset font size |
| `Cmd+N` | New instance |
| `Ctrl+Shift+Space` | Toggle vi mode |
