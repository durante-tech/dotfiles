## How to Use This Course

Work through the levels in order. Each level builds on the previous one. Don't skip ahead — muscle memory needs time to form.

**Pace:** One level per week is ideal. Spend daily practice time on the current level before moving on.

*Goal: Open files, edit text, save, quit. Don't panic.*

| # | Topic | Doc | Time |
|---|-------|-----|------|
| 1.1 | Neovim 5-minute essentials | [Quick Start](/getting-started/quick-start/) | 15 min |
| 1.2 | Modes, movement, basic editing | [First Day](/getting-started/first-day/) | 2-3 hours |
| 1.3 | Daily cheatsheet (keep open!) | [Daily Cheatsheet](/neovim/daily-cheatsheet/) | Reference |
| 1.4 | Terminal basics (Ghostty) | [Ghostty](/ghostty/) | 5 min read |

**Daily practice:**
- Open 5 files with `<leader>pf`
- Edit and save with `i` → edit → `Esc` → `:w`
- Navigate with `w`, `b`, `{`, `}`
- Undo/redo with `u` / `Ctrl-r`

**Checkpoint:** You can open, edit, save, and quit files without googling.

**Checkpoint:** You never hold `j` or `k` to scroll. You search to move.

*Goal: Edit text surgically with operators + text objects.*

| 3.5 | Tmux panes & copy mode | [Panes](/tmux/workflows/panes/), [Copy Mode](/tmux/workflows/copy-mode/) | 20 min |

**Daily practice:**
- Use `ci"`, `di(`, `ya{` (text objects)
- Use `.` (dot command) to repeat edits
- Use `:%s/old/new/gc` for replacement
- Split editor: `:vs` then `Ctrl-h/l`
- Use named registers (`"ayy`, `"ap`)

**Checkpoint:** You think in "verb + object" (e.g., `ciw` = "change inner word") instead of cursor positioning.

## Learning Philosophy

1. **Learn one thing at a time.** Don't try to memorize every keybinding. Add 2-3 per day.
2. **Use it 100 times.** A keybinding isn't learned until it's muscle memory. Force yourself.
3. **Don't fall back to old habits.** If you catch yourself reaching for the mouse, stop. Find the keyboard way.
4. **It gets worse before it gets better.** Week 1 will feel slow. By week 4 you'll be faster than any GUI.
5. **Customize gradually.** Understand the defaults before changing them.
